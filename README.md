# Credit_Card_Financial_Dashboard

1. Project Title

Credit Card Financial Weekly Dashboard – Power BI Project

2. Short Description / Purpose

This project focuses on developing a comprehensive Credit Card Financial Weekly Dashboard using Power BI.
The main objective is to provide real-time insights into key credit card performance metrics such as revenue, transactions, customer activity, activation rate, and delinquency rate.
The dashboard enables stakeholders to monitor weekly trends, compare week-over-week performance, and analyze customer behavior for better decision-making.

3. Tech Stack

-The key technologies used in this project include:
-Microsoft Power BI Desktop
-SQL Database (for data storage and querying)
-Power Query for data transformation
-DAX (Data Analysis Expressions) for calculated measures
-Excel / CSV files (imported into SQL)


4. Data Source

-The dataset consists of:
-Credit card transaction details
-Customer demographic data
-Revenue components (annual fees, transaction amount, interest earned)
-Weekly transaction records
-Data is first imported from CSV files into a SQL database, then connected to Power BI for modeling and visualization.

5. Features / Highlights

-Imported and structured raw CSV data into SQL database
-Created calculated columns such as Age Group and Income Group using DAX
-Built revenue calculations including annual fees, interest earned, and transaction amount
-Implemented Week-over-Week (WoW) revenue comparison
-Designed interactive visuals for revenue, transaction count, and customer trends
-Analyzed customer contribution by gender and card category
-Identified top-performing states contributing to overall revenue
-Tracked activation rate and delinquency rate
-Delivered business insights such as YTD revenue and performance growth
-Built a professional, stakeholder-ready weekly financial dashboard
